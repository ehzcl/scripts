This repository contains personal Python and shell scripts used in my daily Linux environment to automate repetitive tasks and improve workflow efficiency.

These scripts are designed to support a lightweight, keyboard-driven workflow and reduce manual overhead in day-to-day system usage.

## Python utilities

### notes_bundle.py / notes_unbundle.py
These two scripts work as a pair to manage note storage in a single-file format.

- `notes_bundle.py`: packs multiple notes into a single structured file for easier storage and transfer (useful for paste-based or single-file note management)
- `notes_unbundle.py`: restores the original notes from the bundled file

This is designed for simple note portability and organization rather than compression or encryption.

### translate_subs.py
Translates subtitle files using a Google Translate API wrapper (googletrans).

## Shell utilities

- logshutdown: logs system uptime before shutdown for session tracking
- renpy: launches Ren'Py games and ensures executable permissions when needed
- rpgm: launcher for RPG Maker MV games
- sb: adjusts screen brightness
- sort_onetab: sorts OneTab export files by domain

## Purpose

These scripts are part of my personal Linux setup and focus on:

- workflow automation
- system productivity
- reducing repetitive manual tasks
- improving usability of my daily Linux environment

They are not framework-level tools, but practical utilities built for real daily usage.
